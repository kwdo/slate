    • Authentication

    To authorize, use this code:

    # With shell, you can just pass the correct header with each request
curl "http://rest-api.transport-trainer.de"
  -H "X-Auth-Token: meowmeowmeow"

    Make sure to replace meowmeowmeow with your API key.

    This API expects for the API key to be included in all API requests to the server in a header that looks like the following:

    X-Auth-Token: meowmeowmeow

    You must replace meowmeowmeow with your personal API key.

    CourseCategories

    Get All CourseCategories from given timestamp

    curl "http://rest-api.transport-trainer.de/course-category/ids/1519257600"  -H "X-Auth-Token: meowmeowmeow"

    The above command returns JSON structured like this:

    [
    "2280496",
    "2280499",
    "2280967",
    "2280497"
]

    This endpoint retrieves all categories with change dates greater than given timestamp.

    HTTP Request

    GET http://rest-api.transport-trainer.de/course-category/ids/<TIMESTAMP>

    Query Parameters

    Parameter Default Description
    timestamp 0 Unix Timestamp to start with
